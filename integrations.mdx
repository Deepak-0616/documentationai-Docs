## Webhook Setup for Notifications

Set up webhooks to receive events like new episodes or watch completions.

### Create a Webhook

<Steps>
  <Step title="Generate Endpoint" icon="link">
    In your Kakarot dashboard, create a new webhook and copy the URL.
  </Step>
  <Step title="Configure Events">
    Select events: `episode.new`, `watchlist.updated`.
  </Step>
  <Step title="Verify Payload">
    Test with a sample event.
  </Step>
</Steps>

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const express = require('express');
  const app = express();
  app.use(express.json());

  app.post('/kakarot-webhook', (req, res) => {
    console.log('Event:', req.body.event);
    console.log('Data:', req.body.data);
    res.status(200).send('OK');
  });

  app.listen(3000);
  ```
  ```python
  from flask import Flask, request

  app = Flask(__name__)

  @app.route('/kakarot-webhook', methods=['POST'])
  def webhook():
      data = request.json
      print(f"Event: {data['event']}")
      print(f"Data: {data['data']}")
      return 'OK', 200

  app.run(port=3000)
  ```
</CodeGroup>

<Request tabs="cURL" show-lines="true">
  ```bash
  curl -X POST https://your-webhook-url.com/kakarot-webhook \
    -H "Content-Type: application/json" \
    -d '{
      "event": "episode.new",
      "data": {
        "title": "Episode 5",
        "series": "Action Series"
      }
    }'
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "status": "received",
    "timestamp": "2024-01-15T10:30:00Z"
  }
  ```
</Response>

## API Access for Advanced Users

For custom integrations, use the Kakarot API at `https://api.example.com/v1`.

<Expandable title="Authentication" default-open="true">
  Use Bearer tokens.

  <ParamField header="Authorization" param-type="string" required="true">
    `Bearer YOUR_TOKEN`
  </ParamField>
</Expandable>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/episodes', {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN'
    }
  });
  const data = await response.json();
  ```
  ```python
  import requests
  headers = {'Authorization': 'Bearer YOUR_TOKEN'}
  response = requests.get('https://api.example.com/v1/episodes', headers=headers)
  data = response.json()
  ```
</CodeGroup>
