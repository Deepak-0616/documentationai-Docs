- Introduced release calendar integration for upcoming anime schedules.

- Optimized mobile UI for smoother navigation across genres.

## Bug Fixes

- Fixed playback interruptions on low-bandwidth connections.
- Resolved profile sync issues across devices.

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core streaming engine with categories and A–Z browsing.
- User profiles, settings, and 7-day free trial signup.

## Breaking Changes

- Updated API endpoints from `/v0/` to `/v1/`. Migrate by updating your client code:

````javascript
// Old
fetch('https://api.example.com/v0/anime');

// New
fetch('https://api.example.com/v1/anime');
````

## Initial Setup

- Basic authentication with `YOUR_API_KEY`.

</Update>
