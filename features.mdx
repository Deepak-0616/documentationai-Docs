## Personalized Top Picks and Recommendations

Kakarot uses your watch history to generate `{topPicks}` and recommendations. Navigate to the home page to see sections like "New", "Popular", and "Top Picks".

To fetch recommendations programmatically:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/recommendations', {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN'
    }
  });
  const data = await response.json();
  console.log(data.recommendations); // Array of anime objects
  ```
  ```python
  import requests
  response = requests.get(
      'https://api.example.com/v1/recommendations',
      headers={'Authorization': 'Bearer YOUR_TOKEN'}
  )
  data = response.json()
  print(data['recommendations'])  # List of anime suggestions
  ```
</CodeGroup>

<Callout kind="tip">
  Refine recommendations by rating shows and updating your profile preferences.
</Callout>

## Managing Crunchylists and Watch History

Organize your anime journey with Crunchylists. Follow these steps to create and manage lists:

<Steps>
  <Step title="Create a List" icon="plus">
    Click "Crunchylists" in the top menu, then select "New List". Name it, like "Shonen Favorites".
  </Step>
  <Step title="Add Content" icon="book-open">
    Search for an anime, click the `+` icon next to it, and choose your list.
  </Step>
  <Step title="View History" icon="history">
    Access "History" from the profile menu to replay recent watches or clear entries.
  </Step>
  <Step title="Share List" icon="share">
    Open your list and click "Share" to generate a public link.
  </Step>
</Steps>

## Simulcasts, Release Calendars, and Sub/Dub Options

Access live simulcasts under the "Simulcast" tab. Use the release calendar to track upcoming episodes.

Choose your preferred audio and subtitles:

<Tabs>
  <Tab title="Subtitles (Sub)" icon="translate">
    Default for authentic Japanese audio with English subs. Ideal for purists.
    
    Select "Sub" from the playback settings gear icon.
  </Tab>
  <Tab title="Dubs (Dub)" icon="volume">
    English dubbed versions for casual viewing. Availability varies by title.
    
    Switch to "Dub" in the same settings menu.
  </Tab>
</Tabs>
